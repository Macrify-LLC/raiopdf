# Document Properties

Document Properties shows you the facts about the open file: its title and author
information, the number of pages, the page size, the file size, and whether it has
a searchable text layer. It's a read-only view — nothing changes.

## How to do it

In the **Organize** tools, choose **Document Properties**.

## What to know

- **Nothing to save** — it only shows information.
- **"Searchable"** tells you whether the document has real, selectable text. If it
  says no, run **Make Searchable**.
- To remove the hidden author and software details, use **Scrub Metadata**.
