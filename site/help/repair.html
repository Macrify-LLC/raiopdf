Repair

Some PDFs are built in unusual ways and won't open properly. Repair rebuilds the file so it opens cleanly.

How to do it

If a file fails to open, RaioPDF offers to repair it for you. You can also do it yourself:

  1. In the Organize tools, choose Repair....
  2. Click Repair PDF.

What to know

  • It runs on your machine, inside RaioPDF.
  • The repaired version opens as a new file. When you Save, RaioPDF asks for a new file name, so your original is left as it was.
