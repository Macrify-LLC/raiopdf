# Compress

Compress shrinks a large PDF's file size — handy when a file is over a portal's
limit or too big to email.

## How to do it

1. In the **Organize** tools, choose **Compress...**.
2. Set a **Quality** from 1 to 9, and turn on **Grayscale** if color isn't needed.
3. Click **Compress PDF**.

RaioPDF shows the before-and-after size so you can see how much it saved.

## What to know

- **It runs on your machine**, inside RaioPDF — nothing is uploaded.
- **Lower quality** makes a smaller file but a less sharp one; **higher quality**
  keeps it crisp but larger. Try a middle setting first.
- **It changes the open document.** Save to keep it — use **Save As** to keep the
  original.
