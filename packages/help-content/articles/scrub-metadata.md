# Scrub Metadata

Every PDF quietly carries metadata — hidden details about the document, such as
the author's name, the software that made it, and the dates it was created and
changed. It doesn't show on the page, but anyone who opens the file can read it.
**Scrub Metadata** lets you see what's there and remove it.

## How to do it

1. Open the document.
2. In the **Legal** tools, choose **Scrub Metadata**.
3. Look over the table — Title, Author, Creator, Producer, and the created and
   modified dates. Anything not set shows as "Not set."
4. Click **Scrub Metadata** to clear it.

## What to know

- **It doesn't touch the visible pages.** Scrubbing removes the hidden details
  only — the content and any annotations are left exactly as they are.
- **Save to keep the cleaned version.** The change applies to the document you
  have open; use **Save** (or **Save As** for a copy) to write it out.
- **Worth doing before you send a file out.** Producing or filing a document is a
  good moment to check what it's carrying about your office.
