# Crop / Resize

Crop / Resize trims the margins off pages, resizes them to Letter or Legal, or
both — useful for a document that was scanned oversized or with wide borders.

## How to do it

1. In the **Organize** tools, choose **Crop / Resize...**.
2. Set which **Pages** to affect (all, by default).
3. Set a **Crop margin** in inches (0 to 2) to trim off the edges.
4. Choose **Resize pages**: **Keep current size**, **Letter**, or **Legal**.
5. Click **Apply Crop / Resize**.

## What to know

- **The result is a new file.** When you **Save**, RaioPDF asks for a new file
  name, so your original is left untouched.
