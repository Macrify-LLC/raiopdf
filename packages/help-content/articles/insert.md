# Insert from File

Insert from File drops the pages of another PDF into the current document at a
position you pick — handy for slipping a signed page or an exhibit into the middle
of a file.

## How to do it

1. In the **Organize** tools, choose **Insert from File...**.
2. Click **Choose PDF...** and pick the file.
3. Set **Insert after page** — use **0** to put it before page 1.
4. Click **Insert from File**.

## What to know

- **The result is a new file.** When you **Save**, RaioPDF asks for a new file
  name, so your original is left untouched.
